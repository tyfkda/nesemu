for node.js
===========

### Requirement

  * SDL2


### Set up

  1. Install SDL2

, then

```bash
$ npm install
```

### Build

```bash
$ npm run build
```

nesemu.js is generated.

### Execution

```bash
$ npm run exec -- <.nes or .zip>
```

or

```bash
$ node nesemu.js <.nes or .zip>
```


## Control

  * Escape key: Quit app.
